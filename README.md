# 사주 · 관상 — 나를 읽는 재미

서버 없이 브라우저에서만 동작하는 사주·성명학·관상 자기이해 도구입니다.

- 입력한 생년월일·이름·사진은 **어떤 서버로도 전송되지 않습니다** (전부 브라우저 안에서 계산됩니다).
- 절기 절입시각(1900~2100, VSOP87 천문계산), 서머타임·표준시 이력, 진태양시 반영.
- 오락 및 자기이해 목적의 콘텐츠이며, 의학적·법률적·재무적 판단의 근거가 될 수 없습니다.
